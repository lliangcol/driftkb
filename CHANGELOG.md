# Changelog

## 0.1.0 - 2026-06-01

- Add the first local DriftKB CLI release.
- Add `driftkb validate` for curated Markdown KB validation.
- Add constrained `rg` verify blocks with explicit expected assertions.
- Add git-diff stale checks that include committed, staged, unstaged, and
  untracked source changes.
- Add fingerprint snapshots and explicit `driftkb fingerprints update`.
- Add optional static call graph cache warning propagation.
- Add advisory `driftkb gaps detect` and human-reviewed `driftkb promote`.
- Add JSON validation reports and text CLI summaries.
- Add pre-push hook installation that calls `driftkb validate`.
- Add the optional `enterprise-java` profile and adapter.

## Unreleased

- No unreleased changes yet.
