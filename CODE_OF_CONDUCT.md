# Code of Conduct

DriftKB follows the spirit of the Contributor Covenant: be respectful, direct,
and constructive when participating in this project.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect different viewpoints and implementation preferences.
- Focus feedback on the work, the evidence, and the tradeoffs.
- Accept responsibility for mistakes and correct them promptly.

## Unacceptable Behavior

- Harassment, insults, threats, or sustained personal attacks.
- Publishing private information without explicit permission.
- Deliberately derailing issues, pull requests, or security reports.
- Any conduct that would make the project unsafe for contributors or users.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests;
or temporarily or permanently ban participants who violate this code.

Report conduct concerns through the maintainer contact path listed by the
repository. Security vulnerabilities should follow `SECURITY.md`.
